# Security Policy

At present, the project is in **active development** and does not have official versioned releases. Security updates are applied to the **main branch** only. Once stable releases are published, older versions will be marked accordingly.

| Version   | Supported          |
| --------- | ------------------ |
| master    | :white_check_mark: |
| releases  | :x: (none yet)     |

## Reporting a Vulnerability

If you discover a security vulnerability in **Kingfisher Browser**, please report it responsibly:

- **Contact**: Send details to the maintainer at 223002089@student.green.ac.bd [(student.green.ac.bd in Bing)](https://www.bing.com/search?q="mailto%3A223002089%40student.green.ac.bd").  
- **Response Time**: You can expect an acknowledgment within **5 business days**.  
- **Process**:  
  - Your report will be reviewed promptly.  
  - If accepted, a fix will be developed and applied to the main branch.  
  - You will be notified once the fix is released or if the issue is declined (with reasoning).  
- **Confidentiality**: All reports are handled privately until a patch is available.
